PlumePad Version 0.1.0
作者:Angelic Sotware(作成者：砂倉瑞希)
種別:フリーウェア

対応OS：WindowsXP/2003/2008/Vista/7(.NET Framework 4.0以降に対応している環境)
※Windows8での動作検証が出来ていないためWindows8は動作保障外となります。

■概要

  メモ帳よりも機能がある、羽根みたいな軽さを目指しているメモ帳です。


■インストール

  圧縮ファイルに同梱されているファイルを任意のフォルダに
  コピーするだけでインストールは完了します。


■アンインストール

  フォルダごと削除を行えばOKです。(レジストリは使用していません)


■ファイル内容(plumepad010.lzh)

  ・ PlumePad.exe    (PlumePad Version 0.1.0本体)
  ・ readme.txt      (このファイル)

  注意：将来フォルダ、ファイル構成が変わる場合があります
